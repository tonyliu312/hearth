# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## In short

- Be respectful, inclusive, and kind. Disagree about code, not about people.
- No harassment, discrimination, personal attacks, or aggressive comments.
- Assume good intent; communicate clearly.
- If something feels off, raise it privately with the maintainer first.

## Enforcement

Reports of unacceptable behavior may be sent privately by opening a GitHub private security advisory or contacting the maintainer. Reports will be reviewed and acted upon. Possible actions range from a warning to a permanent ban from the project.
